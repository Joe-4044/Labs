## Project Requirements

The course project focuses on game **design** and **programming**.
The following requirements **must** be adhered to, unless you have obtained **permission** from the instructors.

Your game should be created **independently** and not be based upon a **tutorial**, or **copy** explicitly an existing game.
While you are welcome to follow tutorials to help you learn new **functionality** you would like to add, your game should not contain the **results** or **output** from a tutorial.

 1. The project is **team-based** (teams should have 4-5 people). 
    + The game to be developed is a multi-level/scene game with a minimumum of **N levels** (where N is the number of teammates).    

 2. The game should have a **start** (to introduce the game and authors) and **end** **screen** (for winning and/or losing).
    + The result of the project must be a **complete, playable game** with >= N levels, not including the start and end screens.
 
 3. The project and resulting game must include **storytelling** with audio **or** (on screen) text **narration**.
    + This doesn't need to be a **prominent** part of the game, but should be represented.
 
 4. The **project** and resulting **game** must include the following game programming technologies:
    1. 2D **graphics** (with animations) or 3D **graphics**
    2. **Sound**: effects, music, etc.
    3. **Physics**
    4. Dynamic (*or multiple*) **cameras**
    5. **User interface** (Menu, pause screen, etc)
    6. Enemy/world **Ai**. 
       + For AI, you will create some game objects that have **state-based** behaviour and involve **pathfinding**.
         + *Don't worry too much about this for now, until we have covered it in class*
   
 5. Your game should be able to run successfully in a window and in a **16x9 aspect ratio**. 
   
 6. Your game should be in **Godot version 4.4.1** and pushed to the Github Repo Provided.
 
 7. Inlude a **demo video**, showing all the features of your game
      + The video will be submitted via D2l, by **one** member of the group.
